MySQL主键、外键和索引

发表于 2017-09-16 | 分类于 MySQL | | 阅读次数
| 字数统计 877

1. 主键

什么是主键？

主键其实是唯一性索引的一种，我们使用Laravel时，通常会在表里加一个自增的id，默认表中的id为主键。

属性：唯一、不为空。

2. 外键

什么是外键？

表的外键是另一表的主键, 外键可以有重复的, 可以是空值。多表关联时，我们常用外键与主表建立联系，这样在删除数据时就可以不用从代码上关联删除记录，MySQL会自动删除从表记录。当然这需要在创建表时设计好外键约束。
